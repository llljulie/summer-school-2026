## US-001 · P1 · Регистрация по телефону

**Как** новый клиент скалодрома,  
**я хочу** зарегистрироваться по номеру телефона и SMS-коду,  
**чтобы** быстро получить доступ к записи без обращения к администратору.

**Критерии приёмки:**

- [ ] Ввод номера телефона с маской ввода (только цифры и символ `+`)
- [ ] Валидация номера на клиенте перед отправкой
- [ ] При невалидном номере — сообщение: «Введите корректный номер телефона в формате +7 XXX XXX-XX-XX»
- [ ] Приведение номера к формату E.164
- [ ] Получение и ввод SMS-кода
- [ ] Кнопка «Отправить повторно» с таймером 60 секунд
- [ ] При превышении лимита попыток — сообщение: «Слишком много попыток. Попробуйте через 15 минут»
- [ ] Успешная регистрация открывает главный экран приложения
- [ ] Сессия сохраняется между запусками

**Связи:** FR-001, FR-033, UC-001, BR-001

## US-002 · P1 · Мои предстоящие записи

**Как** записавшийся клиент,  
**я хочу** видеть список своих предстоящих тренировок,  
**чтобы** не забыть дату, время и детали занятия.

**Критерии приёмки:**

- [ ] Экран «Мои записи» с датой, временем, форматом, инструктором
- [ ] Актуальный статус брони (активна / отменена клиентом / отменена скалодромом)
- [ ] Для броней «Отменена скалодромом» отображается причина отмены

**Связи:** FR-012, UC-004

## US-003 · P1 · Просмотр свободных слотов

**Как** клиент,  
**я хочу** видеть доступные тренировки на ближайшую неделю,  
**чтобы** выбрать подходящее время и формат.

**Критерии приёмки:**

- [ ] По умолчанию — 7 дней вперёд
- [ ] Слоты отображаются в хронологическом порядке (от ближайшего к самому отдаленному)
- [ ] В карточке слота: время, формат, инструктор, средний рейтинг инструктора, свободные места, цена, адрес
- [ ] Empty state с текстом и кнопкой «Изменить даты», если слотов нет

**Связи:** FR-002, FR-004, FR-005, FR-019, UC-002, BR-005, BR-007

## US-004 · P1 · Запись на тренировку

**Как** клиент,  
**я хочу** записаться на выбранный слот в несколько шагов,  
**чтобы** гарантированно занять место в группе.

**Критерии приёмки:**

- [ ] Путь: слот → снаряжение → подтверждение → «Вы записаны»
- [ ] Запись только на себя
- [ ] При попытке записаться на уже занятый слот — ошибка: «Вы уже записаны на эту тренировку»
- [ ] При отсутствии мест — понятная ошибка, без ложного успеха
- [ ] При таймауте (5 секунд) — сообщение: «Сервер временно недоступен. Пожалуйста, повторите попытку»
- [ ] Отображается итоговая цена с детализацией (тренировка + прокат)
- [ ] Получение push-уведомления о подтверждении записи

**Связи:** FR-006, FR-007, FR-008, FR-009, FR-013, FR-021, FR-030, UC-003, BR-001, BR-002

## US-005 · P1 · Выбор своего или прокатного снаряжения

**Как** клиент без полного комплекта снаряжения,  
**я хочу** при записи указать «со своим» или «нужен прокат»,  
**чтобы** скалодром подготовил нужное оборудование.

**Критерии приёмки:**

- [ ] Выбор обязателен на каждой записи
- [ ] Нельзя подтвердить бронь без выбора снаряжения
- [ ] Выбор отображается в подтверждении и в «Мои записи»
- [ ] При нехватке проката кнопка «Нужен прокат» неактивна с пояснением

**Связи:** FR-007, FR-024, UC-003 (3)

## US-006 · P1 · Отмена записи

**Как** клиент,  
**я хочу** отменить запись,  
**чтобы** освободить место, если планы изменились.

**Критерии приёмки:**

- [ ] Отмена доступна из «Мои записи»
- [ ] При отмене за **> 10 минут** до начала — без штрафа, место освобождается
- [ ] При отмене за **≤ 10 минут** до начала — со штрафом (начисление штрафного балла), место освобождается
- [ ] Перед отменой со штрафом отображается предупреждение с запросом подтверждения
- [ ] В профиле отображается количество штрафных баллов
- [ ] Отображается прогресс очистки штрафных баллов: «До сброса штрафных баллов осталось N успешных посещений»
- [ ] При 3 штрафных баллах запись блокируется на 24 часа с отображением таймера
- [ ] Блокировка не влияет на отмену существующих записей
- [ ] Штрафные баллы сбрасываются после 5 успешных посещений подряд без штрафов

**Связи:** FR-010, FR-011, FR-028, UC-005, BR-010

## US-007 · P1 · Уведомление об отмене скалодромом

**Как** записанный клиент,  
**я хочу** получить уведомление, если тренировку отменили (профилактика и т.п.),  
**чтобы** не приехать зря и понять причину.

**Критерии приёмки:**

- [ ] Push с текстом причины
- [ ] В «Мои записи» статус «Отменена скалодромом» и причина
- [ ] Повторная запись на этот слот недоступна
- [ ] При следующем открытии приложения показывается модальное окно с причиной (если push не доставлен)

**Связи:** FR-014, FR-015, FR-016, UC-006, BR-009

## US-008 · P2 · SMS-напоминания о тренировке

**Как** клиент,  
**я хочу** получать SMS за 24 часа и за 2 часа до тренировки,  
**чтобы** не забыть о записи.

**Критерии приёмки:**

- [ ] SMS за 24 ч с датой, временем, адресом скалодрома
- [ ] SMS за 2 ч до начала
- [ ] Напоминания не отправляются для отменённых броней

**Связи:** FR-017, FR-018, UC-007, BR-006

## US-009 · P2 · Оценка инструктора

**Как** клиент после тренировки,  
**я хочу** поставить инструктору оценку от 1 до 5 звёзд,  
**чтобы** поделиться впечатлением и помочь скалодрому улучшать сервис.

**Критерии приёмки:**

- [ ] Оценка доступна в течение 24 ч после окончания слота
- [ ] Оценка доступна только для посещённых тренировок (поле `attended`)
- [ ] Шкала 1–5 звёзд
- [ ] После отправки оценки кнопка становится неактивной с текстом: «Вы уже оценили эту тренировку»
- [ ] Push-приглашение оценить через 1 час после окончания ведёт на экран оценки
- [ ] Средний рейтинг инструктора отображается в карточке слота до записи

**Связи:** FR-019, UC-008, BR-007

## US-010 · P2 · Статус постоянного клиента

**Как** постоянный клиент скалодрома,  
**я хочу** видеть в приложении свой статус лояльности и преимущества,  
**чтобы** чувствовать ценность регулярных занятий.

**Критерии приёмки:**

- [ ] В профиле отображается счётчик посещений
- [ ] Отображается прогресс: «До скидки осталось X тренировок»
- [ ] Скидка 50% применяется автоматически на 10-е посещение
- [ ] Скидка отображается отдельной строкой на экране подтверждения
- [ ] После применения скидки счетчик продолжает накапливаться (11-я → 1-я, 12-я → 2-я)

**Связи:** FR-020, UC-009, BR-008

## US-011 · P2 · Подтверждение записи уведомлением

**Как** клиент,  
**я хочу** получить push-уведомление после успешной записи,  
**чтобы** быть уверенным, что место закреплено.

**Критерии приёмки:**

- [ ] Push-уведомление отправляется после успешного бронирования
- [ ] Содержит дату, время, формат

**Связи:** FR-021, UC-003 (5a)

## US-012 · P2 · Вход без повторной регистрации

**Как** вернувшийся клиент,  
**я хочу** войти по телефону и SMS-коду,  
**чтобы** быстро попасть к своим записям.

**Критерии приёмки:**

- [ ] Вход по номеру + SMS для существующего аккаунта
- [ ] Сессия сохраняется между запусками
- [ ] При невалидном токене — перенаправление на экран входа

**Связи:** FR-022, FR-027, UC-001 (2a)

**чтобы** спланировать тренировки заранее.

**Критерии приёмки:**

- [ ] Фильтр периода «с / по»
- [ ] Список слотов обновляется по выбранному периоду

## US-016 · P2 · Управление сессией и выход

**Как** клиент,  
**я хочу** иметь возможность выйти из приложения,  
**чтобы** обеспечить безопасность аккаунта при передаче устройства.

**Критерии приёмки:**

- [ ] В профиле есть кнопка «Выйти»
- [ ] При нажатии токен удаляется, клиент переходит на экран входа
- [ ] Смена номера телефона доступна только через администратора

**Связи:** FR-027

## US-017 · P2 · Отображение штрафных баллов

**Как** клиент,  
**я хочу** видеть количество штрафных баллов и статус блокировки,  
**чтобы** понимать последствия частых отмен.

**Критерии приёмки:**

- [ ] В профиле отображается количество штрафных баллов
- [ ] Отображается прогресс: «До сброса штрафных баллов осталось N успешных посещений»
- [ ] При 3 баллах отображается таймер блокировки

**Связи:** FR-028

## US-018 · P2 · История записей

**Как** клиент,  
**я хочу** видеть историю моих прошлых записей,  
**чтобы** отслеживать свои тренировки и оценивать инструкторов.

**Критерии приёмки:**

- [ ] Отдельный раздел «История»
- [ ] Список завершенных и отмененных тренировок
- [ ] Фильтр по дате (период «с / по»)
- [ ] Возможность оценить инструктора для завершенных тренировок (в пределах 24 ч)

**Связи:** FR-031, UC-011

## US-019 · P2 · Настройка уведомлений

**Как** клиент,  
**я хочу** управлять уведомлениями из приложения,  
**чтобы** не получать лишние напоминания.

**Критерии приёмки:**

- [ ] В профиле доступна настройка: включить/отключить все push-уведомления разом
- [ ] Критические уведомления (отмена скалодромом) всегда доставляются независимо от настроек

**Связи:** FR-029

## US-020 · P1 · Обработка ошибок API

**Как** клиент,  
**я хочу** получать понятные сообщения при ошибках,  
**чтобы** понимать, что пошло не так и что делать.

**Критерии приёмки:**

- [ ] Для каждого HTTP-статуса показывается понятное пользователю сообщение
- [ ] Сообщения не содержат технических кодов ошибок

**Связи:** FR-032
